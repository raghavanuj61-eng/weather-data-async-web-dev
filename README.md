# 🌦️ Async Weather Data Fetcher

This project demonstrates how to fetch weather data asynchronously using modern programming techniques. It is designed to be fast, efficient, and scalable when working with multiple API requests.

---

## 🚀 Features

* Asynchronous API requests for faster performance
* Supports multiple locations
* Error handling for failed requests
* Clean and modular code structure
* Easy to extend and customize
